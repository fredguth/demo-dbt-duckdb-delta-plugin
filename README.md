# dbt-duckdb-delta-plugin-demo
An example project for duckdb delta integration

## There are two important steps

- register delta plugin in the profiles.yml as [here](https://github.com/milicevica23/dbt-duckdb-delta-plugin-demo/blob/main/dbt_duckdb_delta/config/profiles.yml)
- define delta tables as the [source tables](https://github.com/milicevica23/dbt-duckdb-delta-plugin-demo/blob/main/dbt_duckdb_delta/models/source.yml)
